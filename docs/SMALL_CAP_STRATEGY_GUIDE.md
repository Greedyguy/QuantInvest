## 📊 투자 가능 종목 예시

```
종목A (2만원): 15주 = 30만원
```

- **2만원 이하**: 약 200~300개 종목

## ⚠️ 중요 리스크

### 1. 높은 변동성
```
10억 포트폴리오: 30개 분산 → 변동성 낮음
100만 포트폴리오: 3~5개 집중 → 변동성 매우 높음
```

**예상 변동성**: 
- 기존 v3: 약 20%
- Small Cap: 약 40~60% (2~3배)

### 2. 유동성 리스크
- **소형주 특성**: 거래량 적음
- **슬리피지**: 주문 시 가격 변동 큼
- **체결 지연**: 매수/매도 즉시 체결 안 될 수 있음

### 3. 높은 거래비용 비중
```
거래당 비용: 0.3% (수수료 0.15% × 2)

예시:
- 10억 거래: 300만원 (0.3%)
- 100만 거래: 3,000원 (0.3%)

비율은 같지만, 작은 자본에서는 체감 큼
```

### 4. 제한된 종목 선택
- **대형주 배제**: 삼성전자, SK하이닉스 등 불가능
- **우량주 제한**: 대부분 중소형주로 제한
- **리스크 집중**: 특정 섹터에 치우칠 가능성

## 💡 실전 운용 팁

### 1. 종목 수 설정
```python
# 보수적 (추천)
n_stocks=5  # 종목당 20만원, 더 안정적

# 공격적
n_stocks=3  # 종목당 33만원, 변동성 큼
```

### 2. 주가 필터 조정
```python
# 더 많은 선택지
max_price=50000  # 5만원 이하 (약 700~900개)

# 더 저렴한 종목만
max_price=30000  # 3만원 이하 (약 400~600개)
```

### 3. 리밸런싱 주기
```python
# 빈번한 거래 (거래비용 ↑, 적응력 ↑)
rebal_days=10

# 긴 주기 (거래비용 ↓, 안정성 ↑)
rebal_days=20
```

### 4. 팩터 가중치
```python
# 안정성 중시
factor_weights={
    'MOM6': 0.30,
    'VOL': 0.40,  # 변동성 낮은 종목 선호
    'QUALITY': 0.20,
    'MOM3': 0.05,
    'VAL': 0.05,
}

# 수익성 중시
factor_weights={
    'MOM6': 0.50,  # 모멘텀 강화
    'MOM3': 0.20,
    'VOL': 0.15,
    'QUALITY': 0.10,
    'VAL': 0.05,
}
```

## 📈 기대 성과

### 낙관적 시나리오
- **CAGR**: 25~35%
- **MDD**: -30~-40%
- **Sharpe**: 0.6~0.8

### 현실적 시나리오
- **CAGR**: 15~20%
- **MDD**: -40~-50%
- **Sharpe**: 0.4~0.6

### 비관적 시나리오
- **CAGR**: 5~10%
- **MDD**: -50~-60%
- **Sharpe**: 0.2~0.4

**주의**: 소액 + 집중투자 = 높은 변동성
